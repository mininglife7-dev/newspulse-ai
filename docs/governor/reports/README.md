# Executive Reports

- **Daily summaries** (`DAILY-YYYY-MM-DD.md`) — written only when meaningful
  change occurred that day.
- **Weekly health reports** (`WEEKLY-YYYY-MM-DD.md`) — always written; health
  scores must cite measurable evidence.

The baseline report lives in [`../executive/`](../executive/). Current
platform status remains canonical in [`../PROJECT_STATE.md`](../PROJECT_STATE.md).
